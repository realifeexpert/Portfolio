## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/realifeexpert/Portfolio.git
cd Portfolio
```

Install dependencies:

```bash
npm install
```

## ▶️ Run Locally

Start frontend and backend together:

```bash
npm run dev
```


## 🏗️ Production Build

Build the application:

```bash
npm run build
```

Run production server:

```bash
npm start
```
